# ETA (Encourage Time for Attention)

장시간 노트북 작업하는 현대인을 위한 딥러닝 기반 집중력 관리 서비스

# 실행방법

```bash
git clone https://github.com/Team-PaceMaker/ETA-client.git
cd ETA-client
npm install
npm run dev
```

> secret key를 별도로 관리하기 때문에 실행을 위해선 config.js 파일 필요함
